In 2010 I thought up an idea to create a sandboxed or contained computing environment within Microsoft Research's Singularity RDK. I have been working on it off and on. I am putting all of my C# and F# projects in this environment and even have a C# Window Manager and File Manager. More on this project later. I'm calling it Singularity+. I'm also working with HomeOS from Microsoft Research. Some new developments are that I'm really not sure if the patch will ever materialize, but if I had my hands on Midori it might. As of now Singularity+ exists only as my workspace under version control. And I've merged the application environment with it. There isn'nt a OS anymore as of July 10, 2016. There might be one day. Maybe if Apple and Microsoft ever merged. Singularity+ is now part of my Windows build lab. This is only a partial listing. Also they should check grades before people can download open source software.
